使用vue编写的音乐播放项目
==
该项目使用到了vue全家桶的vue-router , vuex ,数据获取部分来自网易云音乐的接口，这里没有使用axios，而是使用了自己封装的fecth对象，并用xhr来做兼容
async/await来作为异步写法
--
该项目模仿一位大佬的项目详情可见https://github.com/javaSwing
